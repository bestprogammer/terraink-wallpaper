# Terraink Wallpaper Automation

Automatisches Karten-Wallpaper basierend auf dem aktuellen Standort.
Optimiert für iPhone 13 (1170 × 2532 px).

## Verwendung
iPhone Shortcut → GitHub Actions → PNG generieren → Wallpaper setzen
